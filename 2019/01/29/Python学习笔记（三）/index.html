Python 学习笔记（三）

发表于 2019-01-29

这节知识点较多，所以整篇我用一个文字游戏记录和演示。

Wordgame：

print("文字游戏")
temp = input("猜猜我现在心里想的什么数字？")
guess = int(temp)
if guess == 8:
    print("居然猜对了~~")
    print("哼，猜对也没有奖励")
else:
    print("猜错啦")
print("游戏结束，不玩啦！")

在Python Shell里Ctrl + N 新建一个文本框，将上面的小游戏写进去，写完Run Module 将这个程序跑起来，显示：

看起来很容易，一点都没有游戏的样子。

现在我们就从三个方面来改进这个小游戏，进而达到复习知识点的目的。

  1. 猜错的时候应该给用户相应的提示，例如告诉用户输入结果的值是大了还是小了。
  2. 每运行一次程序，用户只能猜一次，应该提供多次机会给用户猜测。
  3. 每次运行程序时答案可以是随机的。因为程序答案固定，容易导致答案外泄。

让我们先来完成第一个改进计划

条件分支

Python的比较操作符，如下图：

条件成立返回True，不成立返回False

逻辑运算符，如下图：

Python的条件分支语法：

if 条件:
    条件为真（True）执行的操作
else:
    条件为假（False）执行的操作

可以利用Python的条件分支来实现第一个改进计划代码如下：

print("文字游戏 2.0")
temp = input
guess = int(temp)
if guess == 8:
    print("居然猜对了~~")
    print("哼，猜对也没有奖励")
else:
    if guess > 8:
        print("大了，大了~~")
        print("猜错啦！")
    else:
        print("小了。小了~~")
        print("猜错啦！")
print("游戏结束，不玩啦")

第二个改进计划

在编写完文字游戏2.0后，F5把程序跑起来的时候就反映出了一个明显的问题：

程序跑起来一次只能输入一个值，这是非常麻烦的。在我编写完测试的时候跑一次就只能测试一次，值是大了还是小了或者值正确，测试完这些我就要跑三次程序！就上面演示的截图就截了三次。

对于用户来说，体验效果也是不尽人意的，因为只能猜一次，无论猜对了或者猜错了都没有第二次机会。

达到这个目的也不是很难我们只需要在程序里引用一个while即可：

while循环

while的循环语法
    while #后跟一个条件：
        条件为真（True）执行的操作

如果while后跟着的条件成立就会执行True的操作

改进代码如下：

print("文字游戏 2.5")
temp = input("猜猜我现在心里想的什么数字？")
guess = int(temp)
while guess != 8:
    temp = input("不是Ta，请再猜一次叭~")
    guess = int(temp)
    if guess == 8:
        print("居然猜对了~~")
        print("哼，猜对也没有奖励")
    else:
        if guess > 8:
            print("猜错啦！")
            print("大了大了，猜小点~")
        else:
            print("猜错啦！")
            print("小点，往大了猜！")
print("游戏结束，不玩啦")

第三个改进计划之引入外援

random模板

这个random模板里有一个函数叫做randint( )

Ta的作用是返回一个随机的整数

导入random模板

在程序最上面插入 import random
赋值 secret = random.randint(1,10)

改进代码如下：

import random
secret = random.randint(1,10)
print("文字游戏 3.0")
temp = input("猜猜我现在心里想的什么数字？")
guess = int(temp)
while guess != secret:
    temp = input("不是Ta，请再猜一次叭~")
    guess = int(temp)
    if guess == secret:
        print("居然猜对了~~")
        print("哼，猜对也没有奖励")
    else:
        if guess > secret:
            print("猜错啦！")
            print("大了，大了，猜小点~")
        else:
            print("猜错啦！")
            print("小了，小了，往大点猜叭！")
print("游戏结束，不玩啦")
