Python 学习笔记（一）

发表于 2019-01-26

python是一个跨平台语言，什么是跨平台呢。通俗点讲就是你在家里的台式windows系统上做好的Python程序，在学校笔记本上Max系统上一样可以运行，支持的系统还有Linux

那么多语言为什么选择Python呢，Python语言具有什么优点？

  • 人生苦短，我用Python
  • 每个程序员都应学习使用Python或Ruby

有人计算过，Python或Ruby写出的程序的代码行数只相当于相对应的Java代码的行数的五分之一。

如果没有绝对的必要，为什么要花这么多时间写出这么多的代码呢？

而且，有人说，一个优秀的程序员能维护的代码量最多是2万行。这不区分用的语言究竟是汇编，C还是Python/Ruby/PHP/Lisp。

所以，如果你用Python/Ruby写，你一个人干的，不管是干什么，如果换用Java/C/C++，那都需要一个5人的小团队来干。

综上所述，我想学Python

从IDLE启动Python

  • IDLE是个Python Shell，”Shell”的意思是”外壳”，通俗点讲就是一个通过键入文本与程序进行交互的一种途径。

打开IDLE

我们可以看到Python相对应的版本信息：

其次会发现”>>>”的提示符，这个提示符告诉我们Python已经准备好了，正等着我们键入Python指令呢。

与Python的第一次交流

在”>>>”后输入：

print("Hello word!")

显示

Hello word!

双引号” “括起来的是一个字符串，字符串就是文本，也就是我们想让Python显示出来的东西。

在Python中数字的计算

在”>>>”后输入：

print(5+3)

显示

8

如图

直接输入5+3的话，Python是否也会给出同样的答案呢，答案是一定的

那如果输入123456789987654321*987654321123456789这个如此之长的式子呢？！

我来试一下

可见Python同样给出了答案。其实任何一种编程语言都具有运算的能力，只是Python的数字运算没有复杂的长度限制，更便于计算。

字符串与数字相加

在”>>>”后输入：

print("Hello word!" + 5)

显示

Traceback (most recent call last):
File "<pyshell#16>", line 1, in <module>
print("Hello word!" + 5)
TypeError: can only concatenate str (not "int") to str

如图

  • Python为什么会报错

在Python中不能把完全不同的两种东西加在一起，比如说这里的数字和文本，正是这个原因print(“Hello word!” + 5)才会报错。这就好比在说”一个苹果加上五个鸭梨会是多少呢”一样没有多大意义同时也没有这个规定。不过像print(“Hello word!” 5)*这样，乘以一个整数来翻倍就具有一定意义了，这个例子就好比在说把”一个苹果给我翻五倍是多少”答案自然是五个苹果(五个Hello word!)。

课后练习

  • IDLE是什么？

IDLE是一个Python Shell。shell的意思是”外壳”，基本上来讲就是通过键入文本与程序进行交互的途径！像我们windows那个cmd窗口，linux那个黑乎乎的命令窗口，他们都是shell，利用他们我们就可以给操作系统下达命令。同样的我们可以利用IDLE这个shell与Python进行互动。

  • print()的作用是什么？

print( )会在输出窗口中显示一些文本

输出窗口就是IDLE shell

  • Python中表示乘法的符号是什么？

Python中的乘号是*(我们眼里的星号)
